1.　常用标记

1.1　标题

1.1.1　说明

•　使用 # 表示标题，一级标题使用一个 # ，二级标题使用两个 ## ，以此类推，共有六级标题。
•　使用 ===== 表示高阶标题，使用 ——— 表示次阶标题。

2.　==== 和 —- 表示标题时，大于等于2个都可以表示。

1.2　引用

1.2.1　说明

使用 > 表示引用， >> 表示引用里面再套一层引用，依次类推。当 > 与后面文字间添加5个blank时，块注释的文字会有变化。

1.2.3　注意

1.　如果 > 和 >> 嵌套使用的话，从 >> 退到 > 时，两行之间必须要加一个回车或者 > 作为过渡，否则默认为下一行和上一行是同一级别的引用。
2.　引用标记里可以使用其他标记，如：有序列表或无序列表标记，代码标记等。

1.9　链接

1.9.1　说明

Markdown 支持两种形式的链接语法： 行内式和参考式两种形式。

行内式链接使用[](link "Optional title") 表示。其中：
•　[] 内的内容为要添加链接的文字。
•　link 为链接地址。
•　Optional title 为显示标题。显示效果为在你将鼠标放到链接上后，会显示一个小框提示，提示的内容就是 Optional title 里的内容。

参考式链接则使用以下方式，
[Baidu][1]
[1]:www.baidu.com "百度一下，你就知道"，其中：
•　第一个方括号内的内容为要添加链接的文字。
•　第二个方括号里的内容为用以辨识链接的标记。
•　第三个方括号里的内容与第二个对应。
•　冒号后面为链接地址。
•　最后为显示标题。
